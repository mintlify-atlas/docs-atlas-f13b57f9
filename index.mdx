## Overview

The Ethical Audit Framework is a powerful Python-based CLI tool that orchestrates multiple penetration testing attacks and generates professional security reports. It combines network scanning, SQL injection testing, WordPress vulnerability assessment, directory enumeration, and hash cracking into a unified, automated workflow.

## Core Capabilities

The framework provides a rich interactive CLI with six attack modes:

1. **Full Audit** - Complete security assessment (DVWA + WordPress)
2. **Network Discovery** - Auto-discover and attack multiple hosts
3. **Reconnaissance** - Network scanning and directory enumeration
4. **SQL Injection** - Targeted SQLMap attacks with hash cracking
5. **WordPress Brute-Force** - WPScan with rockyou.txt wordlist
6. **Custom Target** - Full audit against a manual IP address

## Risk Analysis

Automatically calculates risk scores based on:

- Critical vulnerabilities discovered
- Exposed dangerous ports
- HTTP services exposure
- Extracted credentials

Risk levels: **Low**, **Medium**, **High**, **Critical**
